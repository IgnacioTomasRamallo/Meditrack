# Meditrack
Nuestro cliente es HealthCore, una clínica médica mediana que atiende a ~500 pacientes por mes. Actualmente manejan todo en planillas de Excel y quieren digitalizar su gestión. Nos contrataron para desarrollar MediTrack, una aplicación web para gestionar:  Pacientes Turnos médicos Historias clínicas básicas Doctores y especialidades
